# Instalação
No Raspberry Pi OS Lite 32-bit ARMv7, conecte tela, SDR e áudio; então execute `sudo ./scripts/install.sh`. O instalador verifica modelo, arquitetura, espaço, rede, SPI/touch e RTL-SDR, pede a saída ALSA e preserva dados existentes. Ao final, ele habilita **e inicia** o serviço, que também será iniciado nos próximos boot. O serviço é do sistema: o autologin não é necessário para ele iniciar. O instalador confirma que o serviço ficou habilitado e ativo antes de concluir. Para auditar sem modificar: `sudo ./scripts/install.sh --dry-run --non-interactive`. Se a tela não abrir, use `radioctl status` e `radioctl logs` antes de reiniciar o serviço. Não rode este instalador nesta máquina Ubuntu/x86: ele recusa corretamente.
