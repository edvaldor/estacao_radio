# Instalação
No Raspberry Pi OS 32-bit ARMv7 com sessão gráfica X11, configure o autologin gráfico do usuário `pi`, conecte tela, SDR e áudio; então execute `sudo ./scripts/install.sh`. O instalador verifica modelo, arquitetura, espaço, rede, SPI/touch e RTL-SDR, pede a saída ALSA e preserva dados existentes. Ao final, instala uma unidade systemd **de usuário** em `/home/pi/.config/systemd/user/`, habilitada para `graphical-session.target`, e a inicia. A aplicação usa X11 com `DISPLAY=:0`, `XAUTHORITY=/home/pi/.Xauthority` e `XDG_RUNTIME_DIR` da sessão; portanto, sem a sessão gráfica/autologin do `pi`, a interface não inicia. O instalador confirma que a unidade ficou habilitada e ativa antes de concluir. Para auditar sem modificar: `sudo ./scripts/install.sh --dry-run --non-interactive`. Se a tela não abrir, use `radioctl status` e `radioctl logs` antes de reiniciar com `radioctl restart`. Não rode este instalador nesta máquina Ubuntu/x86: ele recusa corretamente.
